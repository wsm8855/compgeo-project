Introduction

In this project we will create an interactive pedagogical aid for Ruppert's algorithm.

Description

Ruppert's algorithm is a way to refine Delaunay's triangles. It is able to minimize the amount of triangles in a rendering by eliminating triangles where the circumradius to shortest edge ratio is larger than a predetermined threshold. We will create a 2D visual aid implementing this algorithm.
